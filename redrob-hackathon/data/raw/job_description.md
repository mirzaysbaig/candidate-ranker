# Job Description: Senior Machine Learning Engineer

## Role Overview
We are looking for a Senior Machine Learning Engineer to design, build, and optimize our semantic search and candidate recommendation engine.

## Requirements
- Extensive experience with Python and machine learning frameworks (e.g. PyTorch, TensorFlow).
- Experience building vector search applications using indexers like FAISS.
- Experience building interactive UI dashboards using Streamlit.
- Strong understanding of NLP and embedding models (e.g. Sentence Transformers).
- Minimum 5 years of relevant experience.
